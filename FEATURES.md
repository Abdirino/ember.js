## Feature Flags

* `EMBER_DEFAULT_HELPER_MANAGER`

  Provides a default manager for unrecognized helpers as specified in
  [RFC-756](https://github.com/emberjs/rfcs/blob/master/text/0756-helper-default-manager.md).
